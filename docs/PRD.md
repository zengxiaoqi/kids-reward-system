# 产品需求文档 (PRD) — 小孩行为积分奖惩系统

> 版本: v0.2
> 日期: 2026-06-24
> 状态: 需求调研

---

## 1. 产品概述

### 1.1 产品定位
面向 3-12 岁儿童家庭的 **AI 驱动行为习惯养成工具**。通过积分激励机制和 AI 智能分析，帮助孩子在日常行为中建立良好习惯，同时为家长提供科学的教育建议。

### 1.2 目标用户
- **主要用户**: 家长（管理规则、记录积分、接收 AI 建议）
- **次要用户**: 孩子（查看积分、兑换礼品、与 AI 互动）

### 1.3 核心价值
- 让好习惯变得"看得见"
- 用正向激励替代说教
- **AI 赋能个性化教育**
- **自然语言交互，零学习成本**

## 2. 功能需求

### 2.1 行为管理模块

#### 2.1.1 固定日常行为（系统预设）
每日循环的基础行为项，可启用/禁用、调整积分值：

| 行为项 | 默认积分 | 说明 |
|--------|----------|------|
| 🌅 早上起床洗漱 | +2 | 按时完成晨间洗漱 |
| 🥣 早餐 | +1 | 好好吃完早餐 |
| 📚 学习/作业 | +3 | 完成当日学习任务 |
| 🍱 中午吃饭 | +1 | 按时吃完午餐 |
| 🍽️ 晚上吃饭 | +1 | 按时吃完晚餐 |
| 🪥 晚上洗漱 | +2 | 按时完成晚间洗漱 |
| 😴 睡觉 | +2 | 按时上床睡觉 |

#### 2.1.2 自定义行为（家长添加）
家长可随时新增行为项：

**加分行为示例：**
| 行为 | 积分 | 分类 |
|------|------|------|
| 📖 课外阅读 30 分钟 | +3 | 学习 |
| 🏃 运动/锻炼 | +3 | 健康 |
| 🧹 帮忙做家务 | +2 | 品德 |
| 🎹 练习乐器 | +2 | 学习 |
| 🤝 礼貌待人/分享 | +1 | 品德 |
| 🎨 画画/手工 | +2 | 创意 |
| ✅ 考试进步 | +10 | 学习 |
| 🌟 特别表扬 | 自定义 | 综合 |

**扣分行为示例：**
| 行为 | 积分 | 说明 |
|------|------|------|
| 😤 发脾气/哭闹 | -2 | 情绪管理 |
| 📱 超时使用电子设备 | -3 | 自律 |
| ❌ 说谎 | -5 | 品德 |
| 🗑️ 乱扔东西 | -1 | 习惯 |
| ⏰ 迟到/拖延 | -1 | 时间管理 |
| 😠 打人/骂人 | -5 | 品德 |
| 🍬 偷吃零食 | -1 | 健康 |
| 📺 没完成作业就看电视 | -2 | 学习 |

#### 2.1.3 行为规则配置
- 每个行为项可设置：名称、图标、积分值（正/负）、是否每日重复、时间段限制
- 行为项可分组：学习类、生活类、品德类、运动类、健康类等
- 支持设置每日积分上限（防止刷分）
- 支持设置连续完成奖励（连续 7 天完成某行为额外奖励 +5）
- **积分永不过期**，持续累积

### 2.2 积分记录模块

#### 2.2.1 每日打卡
- 家长手动记录：点击行为项 → 确认完成/未完成
- 快速操作：一键打卡当日所有固定行为
- 支持补录：可以回溯修改过去几天的记录
- 支持撤销：误操作可撤回
- **自然语言打卡**（通过飞书/微信）：
  - "小明今天按时睡觉了"
  - "小红今天帮妈妈洗碗"
  - "小明今天发脾气了"

#### 2.2.2 积分流水
- 每笔积分变动都有记录：时间、行为、积分值、备注
- 可查看每日/每周/每月积分汇总
- 积分变动通知（可选推送给家长/孩子）

### 2.3 礼品兑换模块

#### 2.3.1 礼品管理
家长设置可兑换的礼品及所需积分：

**礼品示例：**
| 礼品 | 所需积分 | 分类 |
|------|----------|------|
| 看动画片 30 分钟 | 10 | 娱乐 |
| 选一个小零食 | 15 | 食物 |
| 去公园玩 | 30 | 外出 |
| 买一本新书 | 50 | 学习 |
| 买一个玩具（50元内） | 100 | 礼物 |
| 去游乐园 | 200 | 外出 |
| 许愿卡（任意合理愿望） | 500 | 特别 |

#### 2.3.2 兑换流程
1. 孩子选择心仪的礼品
2. 确认积分是否足够
3. 家长确认兑换 → 扣除积分
4. 记录兑换历史
5. 支持"心愿单"功能 — 积分不够时先加入心愿单
6. **自然语言兑换**：
   - "小明想兑换去游乐园"
   - "小红要用 50 积分换一本书"

### 2.4 🤖 AI Agent 智能助手（核心功能）

#### 2.4.1 AI 行为分析
- **每日/每周行为报告**：自动分析孩子的行为模式
- **趋势识别**：发现行为习惯的变化趋势
- **问题预警**：连续扣分或某项行为持续未完成时提醒家长
- **亮点发现**：发现孩子的进步和闪光点

#### 2.4.2 AI 教育建议
- **个性化建议**：根据孩子的行为数据给出针对性的教育建议
- **正向引导话术**：教家长如何用正确的方式表扬/引导孩子
- **习惯养成策略**：针对薄弱环节提供改进方案
- **年龄适配建议**：根据孩子年龄调整建议内容

#### 2.4.3 AI 互动对话
- **孩子可以问 AI**：
  - "我还有多少积分？"
  - "我怎么才能换到游乐园？"
  - "我今天表现怎么样？"
- **家长可以问 AI**：
  - "小明最近表现怎么样？"
  - "小红在哪些方面需要改进？"
  - "怎么帮助孩子养成阅读习惯？"
  - "给小明写一段表扬的话"

#### 2.4.4 AI 主动关怀
- 孩子连续表现好时，AI 主动发送鼓励
- 孩子遇到挫折时，AI 给家长建议如何安慰引导
- 重要节点（如连续打卡 7 天）AI 生成专属祝贺

### 2.5 💬 多渠道接入

#### 2.5.1 飞书机器人
- **家长端功能**：
  - 自然语言打卡："小明今天按时起床了"
  - 查询积分："小明现在多少积分？"
  - 兑换礼品："小明要换一本新书"
  - 查看报告："看看小明这周的表现"
  - 咨询建议："怎么让孩子更爱运动？"
- **消息推送**：
  - 每日打卡提醒
  - 积分变动通知
  - AI 分析报告

#### 2.5.2 微信公众号/小程序
- 与飞书端功能对等
- 利用微信生态的便利性
- 支持家长群分享（可选）

#### 2.5.3 App（Flutter）
- 完整的可视化界面
- 数据统计图表
- 礼品商城展示
- 成就系统展示
- 设置管理

### 2.6 数据统计模块

#### 2.6.1 仪表盘
- 当前总积分
- 今日获得积分
- 本周/本月积分趋势图
- 连续打卡天数
- 距离下一个礼品还差多少积分

#### 2.6.2 行为分析
- 各项行为完成率统计
- 最常加分/扣分的行为
- 每周/每月行为报告
- 习惯养成进度（某行为连续完成 N 天标记为"已养成"）

#### 2.6.3 成就系统
- 🏅 连续打卡 7 天 — "一周小达人"
- 🏆 连续打卡 30 天 — "月度之星"
- ⭐ 累计 100 积分 — "百分小能手"
- 🎖️ 兑换第一个礼品 — "梦想启程"
- 💎 累计 1000 积分 — "积分大王"
- 🌈 连续 30 天无扣分 — "品德标兵"

### 2.7 多孩管理

- 支持添加多个孩子档案
- 每个孩子独立的积分账户和行为记录
- 可为不同孩子设置不同的行为规则和礼品
- 孩子之间的积分独立，不做比较
- 家长可对比查看（可选，用于发现每个孩子的特点而非比较）

### 2.8 家长设置

- 行为规则自定义（增删改）
- 积分值调整
- 礼品库管理
- 每日打卡提醒时间设置
- 数据导出（Excel/PDF 报告）
- AI 功能开关和偏好设置

## 3. 非功能需求

### 3.1 用户体验
- 界面友好、色彩丰富、适合儿童审美
- 操作简单，家长 3 分钟内能上手
- 孩子能看懂积分和礼品界面
- 有趣的动画反馈（获得积分时的特效）
- **飞书/微信端零学习成本，发消息即用**

### 3.2 AI 能力要求
- 响应时间 < 3 秒
- 建议内容安全、正向、适合儿童教育
- 支持上下文理解（记住当前讨论的孩子）
- 支持多轮对话

### 3.3 数据安全
- 用户数据加密存储
- 不收集不必要的个人信息
- AI 对话内容加密处理
- 支持数据导出和删除

### 3.4 平台支持
- **飞书机器人**（优先）
- **微信公众号/小程序**
- **Flutter App**（Android/iOS）

### 4.2 技术选型

| 模块 | 技术方案 | 说明 |
|------|----------|------|
| **前端 App** | Flutter | 跨平台，与灵犀伴学技术栈一致 |
| **后端服务** | Node.js + Express / NestJS | 或 Python + FastAPI |
| **数据库** | PostgreSQL | 主数据存储 |
| **缓存** | Redis | 会话缓存、积分计算 |
| **AI 模型** | OpenAI API / Claude / 本地模型 | 行为分析、建议生成 |
| **飞书对接** | 飞书开放平台 Bot API | 消息收发 |
| **微信对接** | 微信公众号 API / 企业微信 | 消息收发 |
| **消息队列** | RabbitMQ / Redis Queue | 异步任务处理 |
| **部署** | Docker + 阿里云/腾讯云 | 容器化部署 |

#### 4.3.2 对话示例

**家长与 AI 对话：**
```
家长：小明最近表现怎么样？
AI：小明这周表现不错！总积分 +25 分。
   📊 亮点：
   - 连续 5 天按时睡觉，睡眠习惯正在养成
   - 主动帮忙做家务 3 次，值得表扬
   
   ⚠️ 需要关注：
   - 有 2 次因为看电视超时被扣分
   - 学习任务完成度 60%，比上周有所下降
   
   💡 建议：
   可以试试"先完成作业再看电视"的规则，
   把看电视作为完成学习的奖励。

家长：怎么跟小明说这件事？
AI：你可以这样说：
   "小明，妈妈看到你最近睡觉都很准时，真棒！
   如果看电视也能像睡觉一样准时，
   妈妈会更开心的。我们一起试试好不好？"
```

**孩子与 AI 对话：**
```
孩子：我还有多少积分？
AI：小红，你现在有 85 积分！
   距离"去游乐园"(200积分)还差 115 积分。
   加油！你可以的！💪

孩子：我怎么才能更快攒到积分？
AI：小红，给你几个小建议：
   1. 每天坚持课外阅读 30 分钟 (+3分)
   2. 帮忙做家务 (+2分)
   3. 连续 7 天完成所有日常任务会有额外奖励 +5分！
   
   你已经连续 3 天按时睡觉了，继续保持！
```
