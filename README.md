# 🌟 小孩行为积分奖惩系统 (Kids Reward System)

通过积分激励机制，帮助小孩养成良好日常行为习惯。

## 项目简介

这是一个面向家庭的儿童行为管理工具，家长可以为孩子的日常行为设定积分规则，孩子通过完成日常任务积累积分，并用积分兑换心仪的礼品奖励。

## 核心功能

- 📋 **日常行为管理** — 管理每天固定和自定义的行为项目
- ⭐ **积分加减** — 根据行为表现自动增减积分
- 🎁 **礼品兑换** — 积分兑换礼品商城
- 📊 **数据统计** — 行为记录和积分趋势分析
- 👨‍👩‍👧 **多孩支持** — 支持多个孩子的独立积分账户
